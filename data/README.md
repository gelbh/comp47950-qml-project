# Data

Dataset and derived files for the COMP47950 QML project (Nim game).

- **`raw/`** — Reserved (Nim states are generated programmatically from code with fixed `random_state`).
- **`processed/`** — Optional cached datasets (e.g. `nim_states_k3_M7.csv`) for convenience and visualisation. Canonical versions are generated on-the-fly by `qml_project.nim.data`.
